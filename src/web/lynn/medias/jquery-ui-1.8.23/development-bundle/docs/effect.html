jQuery UI effect

Overview

[edit]

effect( effect, [options], [speed], [callback] )

Uses a specific effect on an element (without the show/hide logic).

Dependencies

  • Effects Core

Example

  • Demo
  • View Source

Apply the effect explode if you click on the element.

$("p").click(function () {
      $("div").effect("explode");
    });
